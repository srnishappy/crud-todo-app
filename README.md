1. เปิด terminal หรือ command prompt แล้วไปที่โฟลเดอร์ client ของโปรเจค (เช่น crud-todo-app-main/client).
2. รันคำสั่งนี้เพื่อติดตั้ง vite
   npm install --save-dev vite
3.พอติดตั้งเสร็จแล้ว ลองรันคำสั่งนี้อีกรอบ
   npm run dev
